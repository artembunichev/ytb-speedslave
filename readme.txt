ytb-speedslave - opposite of YouTube "speedmaster".

Google Chrome extension for disabling YouTube "speedmaster" 'feature'.



What Does It Do:

This extension disables YouTube 'feature', that runs video at 2x speed when you hold down LMB, and emulates habitual behavior.



Installation:

1. Clone this repo:
	git clone https://github.com/artembunichev/ytb-speedslave

2. Open "chrome://extensions/" turn on "Developer mode", click "Load unpacked" and select directory with ytb-speedslave extension.
